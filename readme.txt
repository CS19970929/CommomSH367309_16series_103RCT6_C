1，以后写代码，函数返回值只会返回监控结果数值。秉承三层设计理念
	A，第一层，底层，返回底层函数处理结果，成功或者失败
	B，第二层，初级应用层，基于底层代码再写一层，这层监控各种底层函数是否运行成功并记录各种错误，返回一个特定的错误类型(因为基本是相互关联的，所以第一个错立刻返回就好)。
	C，第三层，他人使用层，基于第二层代码的使用，原则是void类型返回，如果实在有一些需求则灵活使用返回值。
